# Show Univ FE
Show Univ 프로젝트에서 백엔드 로직과 프론트엔드 로직을 개별 WAS로 분리하여 API통신을 통해 기능을 구현하는 프로젝트입니다.

# Features
- 사용자 인증/인가
- WSYIWYG 에디터
- 첨부파일 업로드
- Kakao Map 기반 위치지정, 공유
- 댓글기능
- 좋아요기능
